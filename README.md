# dotfiles

## Installation

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

git clone https://github.com/jeboehm/dotfiles.git ~/Documents/dotfiles
ln -s ~/Documents/dotfiles/.zshrc ~/.zshrc
```
